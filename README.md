deflow
======

Automatically fetches new wallpapers for you.

Available for OS X v10.6 and later.

* Selectable intervals
* Selectable screen


## Setup

  $ sudo motion update
  $ bundle
  $ pod setup
  $ bundle exec rake pod:install
